Welcome to my very extensive library.

Books

To see all books, check out http://localhost:8080/books

To see one book and it's author, just add an ID to the URL above, for example: http://localhost:8080/books/3

Authors

To see all authors, check out http://localhost:8080/authors

To see one author, just add an ID to the URL above, for example: http://localhost:8080/authors/3

Members

To see all members, check out http://localhost:8080/members

To see one member and all his/her borrowed books, just add an ID to the URL above, for example: http://localhost:8080/members/5

Reset

To reset all member's Late fee, just click this link http://localhost:8080/reset
